
# 🎓 Student Grade Calculator

A console-based Java application that calculates a student's grade
based on marks entered for multiple subjects.
Built as part of my **CodSoft Internship Task**.

---

## 📌 About The Project

This program takes marks obtained (out of 100) in each subject,
calculates the total marks and average percentage,
and assigns a grade with remarks based on the result.

---

## ⚙️ How It Works

1. User enters their **name**
2. User enters the **number of subjects**
3. User enters **subject name and marks** for each subject
4. Program calculates:
   - ✅ Total Marks
   - ✅ Average Percentage
   - ✅ Grade
   - ✅ Remarks
5. A formatted **Result Card** is displayed

---

## 📊 Grade Scale

| Percentage  | Grade | Remark               |
|-------------|-------|----------------------|
| 90 - 100%   | A+    | Outstanding          |
| 80 - 89%    | A     | Excellent            |
| 70 - 79%    | B     | Very Good            |
| 60 - 69%    | C     | Good                 |
| 50 - 59%    | D     | Average              |
| 40 - 49%    | E     | Pass                 |
| 0  - 39%    | F     | Fail                 |
